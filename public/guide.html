캐릭터 레퍼런스 이미지와 텍스트만으로
웹툰 스타일 장면을 AI가 자동 생성합니다

시작하기

✨ 주요 기능

🎨

캐릭터 이미지 생성

캐릭터 레퍼런스를 기반으로 텍스트 설명만 입력하면 일관된 스타일의 새로운 장면을 생성합니다.

🖌️

스케치 변환

러프 스케치를 업로드하면 캐릭터의 아트스타일로 깔끔하게 변환합니다.

✏️

이미지 편집

기존 생성 이미지를 선택하고 수정 사항을 텍스트로 지시하면 편집됩니다.

🖼️

배경 생성

사진을 업로드하면 사람/물체 제거 → 카툰 스타일 변환 → 다양한 앵글 생성까지 자동 처리합니다.

🛒

캐릭터 마켓플레이스

다양한 캐릭터를 바나나로 구매하여 나만의 캐릭터 라인업을 구성하세요.

❤️

My Library

생성된 모든 이미지를 한눈에 관리. 즐겨찾기로 마음에 드는 이미지를 빠르게 찾으세요.

🚀 사용 방법

1

캐릭터 선택

왼쪽 사이드바에서 사용할 캐릭터를 선택합니다. 마켓에서 새 캐릭터를 구매할 수도 있습니다.

2

생성 모드 선택

텍스트(프롬프트만), 스케치(러프 스케치 변환), 편집(기존 이미지 수정) 중 선택합니다.

3

배경 선택 (선택)

드롭다운에서 텍스트 배경을 선택하거나, 배경 탭에서 만든 이미지 배경을 사용할 수 있습니다.

4

프롬프트 입력 & 생성

원하는 장면을 설명하고 "이미지 생성" 버튼을 클릭하세요. AI가 캐릭터를 그려줍니다!

💡 3가지 생성 모드

📝

텍스트 모드

텍스트 설명만으로 캐릭터 장면을 생성합니다. 가장 간편한 방법입니다.

✍️

스케치 모드

손그림이나 러프 스케치를 업로드하면 캐릭터 스타일로 변환합니다.

🔧

편집 모드

이전에 생성한 이미지를 선택하고 수정 지시를 내려 편집합니다.

🛒 마켓플레이스

캐릭터를 골라 나만의 라인업을 구성하세요

마켓플레이스에서 다양한 캐릭터를 바나나로 구매할 수 있습니다. 구매한 캐릭터는 캐릭터 선택 목록에 추가됩니다.

바나나란?

바나나는 워니의 나노바나나봇에서 사용하는 화폐입니다. 이미지를 생성하거나 캐릭터를 구매할 때 사용됩니다. 매월 티어에 따라 무료 생성 횟수가 제공되며, 초과 시 바나나가 차감됩니다.
